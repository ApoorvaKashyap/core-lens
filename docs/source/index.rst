CoRE Lens Library
=======================


A Python library for querying, analysing, and visualising geospatial Earth science data built on CoreStack. Provides a unified interface over microwatershed hydrology, administrative boundaries, and pluggable domain entities — with lazy Parquet I/O, seasonal time-series support, and spatial statistical analysis.
