.. py:function:: parquet(result: core_lens.base.result.Result, path: str | pathlib.Path, **kwargs: Any) -> None

   Export the result data to a standard Parquet file.

   This function uses Polars to write the underlying data frame.

   :param result: The Result object to export.
   :type result: Result
   :param path: Destination path for the Parquet file.
   :type path: str | pathlib.Path
   :param \*\*kwargs: Additional options to pass to Polars `write_parquet`.
   :type \*\*kwargs: Any

   :raises TypeError: If the Result object has geometry.

   .. rubric:: Example

   >>> from core_lens.export import parquet
   >>> parquet(result, "output.parquet", compression="zstd", compression_level=3)

.. py:function:: geoparquet(result: core_lens.base.result.Result, path: str | pathlib.Path, **kwargs: Any) -> None

   Export the result data to a GeoParquet file.

   This function uses DuckDB to write the data frame with spatial extensions.

   :param result: The Result object to export.
   :type result: Result
   :param path: Destination path for the GeoParquet file.
   :type path: str | pathlib.Path
   :param \*\*kwargs: Additional options to pass to DuckDB's COPY statement.
   :type \*\*kwargs: Any

   .. rubric:: Example

   >>> from core_lens.export import geoparquet
   >>> geoparquet(
   ...     result.with_geometry(),
   ...     "output.geoparquet",
   ...     compression="ZSTD",
   ...     partition_by="year"
   ... )

   :raises TypeError: If the Result object does not have geometry.
