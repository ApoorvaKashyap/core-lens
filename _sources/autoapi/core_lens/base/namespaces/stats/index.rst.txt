.. py:class:: StatsNamespace(result: core_lens.base.result.Result)

   Statistical methods on ``result.stats.*``.

   All methods return a fresh :class:`~core_lens.base.result.Result`.
   Computed values always go in ``data``; method parameters go in ``metadata``.


   .. py:method:: describe(columns: list[str] | None = None, by: str = 'column') -> core_lens.base.result.Result

      Per-column or per-entity descriptive statistics.

      :param columns: Numeric columns to describe. Defaults to all numeric cols.
      :type columns: list[str] | None, optional
      :param by: ``"column"`` (one row per column, design default) or
                 ``"entity"`` (one row per entity with mean/std per column).
      :type by: str, optional

      :returns: New :class:`~core_lens.base.result.Result` with descriptive stats.
      :rtype: Result

      :raises ValueError: If ``by`` is not ``"column"`` or ``"entity"``.



   .. py:method:: correlate(columns: list[str], method: CorrelateMethod = CorrelateMethod.PEARSON, across: str = 'entity') -> core_lens.base.result.Result

      Pairwise correlations between columns.

      :param columns: At least 2 column names.
      :type columns: list[str]
      :param method: A :class:`~core_lens.base.namespaces.stats.CorrelateMethod` enum value.
      :type method: CorrelateMethod, optional
      :param across: ``"entity"`` or ``"time"`` — recorded in metadata only.
      :type across: str, optional

      :returns: Result whose data has columns
                ``column_a | column_b | correlation | p_value``.
      :rtype: Result

      :raises CorrelationError: If fewer than 2 columns supplied.
      :raises ValueError: If ``method`` is not recognised.



   .. py:method:: test(column: str, groups: str | None = None, periods: list[tuple[int, int]] | None = None, against: float | None = None, method: TestMethod | None = None, significance_level: float = 0.05) -> core_lens.base.result.Result

      Hypothesis test in three modes: group-based, period-based, single-sample.

      :param column: Numeric column to test.
      :type column: str
      :param groups: Categorical column to split groups on.
      :type groups: str | None, optional
      :param periods: List of ``(from_year, to_year)`` period tuples.
      :type periods: list[tuple[int, int]] | None, optional
      :param against: Reference value for a one-sample test.
      :type against: float | None, optional
      :param method: A :class:`~core_lens.base.namespaces.stats.TestMethod` enum value.
                     Auto-selected via Shapiro-Wilk if ``None``.
      :type method: TestMethod | None, optional
      :param significance_level: Alpha level for ``significant`` flag (default 0.05).
      :type significance_level: float, optional

      :returns: Result whose data has per-group descriptive stats
                ``group | n | mean | std | median`` and
                ``metadata`` with ``statistic``, ``p_value``, ``significant``.
      :rtype: Result

      :raises ValueError: If none of ``groups``, ``periods``, ``against`` supplied,
          or if ``method`` is not recognised.



   .. py:method:: change(column: str, from_period: int, to_period: int, method: ChangeMethod = ChangeMethod.ABSOLUTE) -> core_lens.base.result.Result

      Change between two time periods per entity.

      :param column: Value column to compute change for.
      :type column: str
      :param from_period: Start year/period integer.
      :type from_period: int
      :param to_period: End year/period integer.
      :type to_period: int
      :param method: A :class:`~core_lens.base.namespaces.stats.ChangeMethod` enum value.
      :type method: ChangeMethod, optional

      :returns: For *absolute* / *percentage*: data has
                ``key_col | value_from | value_to | change | pct_change``.
                For *trend*: ``key_col | slope | r_squared | direction``.
      :rtype: Result

      :raises ValueError: If ``method`` is not recognised or year column absent.



   .. py:method:: anomaly(column: str, mode: str, method: AnomalyCrossMethod | AnomalyTsMethod, baseline: tuple[int, int] | None = None, threshold: float = 2.0) -> core_lens.base.result.Result

      Anomaly detection in cross-sectional or timeseries mode.

      :param column: Value column to analyse.
      :type column: str
      :param mode: ``"cross_sectional"`` or ``"timeseries"``.
      :type mode: str
      :param method: An :class:`~core_lens.base.namespaces.stats.AnomalyCrossMethod` or :class:`~core_lens.base.namespaces.stats.AnomalyTsMethod` enum value.
      :type method: AnomalyCrossMethod | AnomalyTsMethod
      :param baseline: ``(from_year, to_year)`` inclusive.  Required for
                       timeseries; optional for cross-sectional.
      :type baseline: tuple[int, int] | None, optional
      :param threshold: Sigma / score threshold for anomaly flag (default 2.0).
      :type threshold: float, optional

      :returns: Result whose data has ``key_col | anomaly_score | is_anomaly``
                (cross-sectional) or ``key_col | year | anomaly_score | is_anomaly``
                (timeseries, baseline period excluded).
      :rtype: Result

      :raises ValueError: If ``mode``, ``method``, or observation count invalid.



   .. py:method:: similarity(target: str, columns: dict[str, Any], method: SimilarityMethod = SimilarityMethod.EUCLIDEAN, top_n: int = 10) -> core_lens.base.result.Result

      Find entities most similar to ``target`` across ``columns``.

      :param target: Key value of the reference entity.
      :param columns: Mapping ``{column_name: None | (resolution, filter_dict)}``.

                      * ``None`` — use the column directly from ``result.data``.
                      * ``(resolution, filter_dict)`` — fetch from the entity's file at
                        the given resolution (``"static"``, ``"annual"``,
                        ``"fortnightly"``).  Supported ``filter_dict`` keys:

                        - ``"year": int | (int, int)`` — year equality or range.
                        - ``"season": str`` — season label (``"kharif"`` etc.).
                        - ``"agg": str`` — time-aggregation before joining
                          (``"mean"`` default, ``"sum"``, ``"min"``, ``"max"``).
      :param method: Distance metric. A :class:`~core_lens.base.namespaces.stats.SimilarityMethod` enum value.
      :param top_n: Number of most-similar entities to return (default 10).

      :returns: Result with ``key_col | similarity_score | rank``.

      :raises ValueError: If ``method`` is invalid, no columns can be resolved,
          or ``target`` is not found.
