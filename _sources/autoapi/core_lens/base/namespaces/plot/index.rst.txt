.. py:class:: SubplotOn(*args, **kwds)

   Bases: :py:obj:`enum.Enum`


   Temporal dimensions to split plot data across.

   Used to specify how data should be partitioned into subplots or map layers
   based on temporal columns added during materialisation.

   :ivar YEAR: Split data by year.
   :ivar MONTH: Split data by month.
   :ivar SEASON: Split data by meteorological season.
   :ivar SEASON_YEAR: Split data by season and year.

   .. py:method:: choropleth(column: str, subplot_on: SubplotOn | None = None) -> lonboard.Map

      Render an interactive choropleth map using Lonboard.

      If the Result does not already have geometry, it will be attached
      automatically via :meth:`~core_lens.base.result.Result.with_geometry`.

      :param column: The column to use for colour mapping.
      :type column: str
      :param subplot_on: Optional temporal dimension to split data across.
                         A :class:`~core_lens.base.namespaces.plot.SubplotOn` enum value. When set, one layer is rendered per unique
                         value of ``subplot_on`` in the data.
      :type subplot_on: SubplotOn | None, optional

      :returns: A Lonboard Map object.
      :rtype: lonboard.Map

      :raises ValueError: If ``column`` or ``subplot_on`` column not found in data.
      :raises NotImplementedError: Lonboard does not support native subplot grids;
          only single-value ``subplot_on`` is rendered when specified.



   .. py:method:: timeseries(x: str | None = None, y: str | list[str] | None = None, subplot_on: SubplotOn | str | None = None, top_n: int = 10, aggregate: bool = False) -> Any

      Render a timeseries line chart using Bokeh.

      Returns a :class:`bokeh.models.Tabs` widget with two panels:
      *Per Entity* (up to ``top_n`` entities) and *Aggregated Mean*.
      When ``aggregate=True``, only the aggregated panel is returned as a
      plain :class:`bokeh.plotting.figure`.

      :param x: The temporal column to plot on the x-axis.
      :type x: str | None, optional
      :param y: The value column(s) to plot on the y-axis.
                If ``None``, all numeric columns (except ``x``) are used.
      :type y: str | list[str] | None, optional
      :param subplot_on: Optional temporal dimension to split data across.
                         A :class:`~core_lens.base.namespaces.plot.SubplotOn` enum value or string.
      :type subplot_on: SubplotOn | str | None, optional
      :param top_n: Maximum number of entities rendered in per-entity view.
      :type top_n: int, optional
      :param aggregate: If ``True``, renders only the aggregated mean view.
      :type aggregate: bool, optional

      :returns: A Bokeh Tabs object (or a single Figure when ``aggregate=True``).
      :rtype: Any
